Description

This is a simulation of natural selection using Neural Networks with Predator, and Prey entities. I was inspired by This Youtube video. An entity's field of vision is divided up into "bins" where each vision bin corresponds to an NN input ("sensory inputs" to their "brain"). Vision input activations are determined by the ratio of the distance from the closest entity in that bin to the max vision range. Clicking on an entity will show its field of vision.

Rules

Prey will live indefinitely unless eaten by a Predator, after a specified amount of time staying alive they are able to reproduce (meaning the simulation will copy it, I'm to lasy to implement sexual reproduction). Predators lose health over time and faster when they move, when they eat a prey they're health goes up and they can reproduce (same as prey). Whenever an entity is reproduced it's offspring's NN is randomly modified to simulate genetic mutation.

Neural Networks

A Neural Network (NN) is basically an entity's brain with nodes "Neurons" and connections "Synopsis" (This is a pretty good explanation). There are input nodes corresponding to vision bins and output nodes corresponding to movement (speed, angular speed), there are no hidden layers. Nodes have a bias and activation function and Connections have a weight and dynamic weight. All of these variables control how the NN "thinks" and are randomly changed when an entity is copied/reproduced.
